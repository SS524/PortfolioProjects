Application Workflow

1. Run the Application 
2. User will see a signup page to enter his/her name 
3. When user will click on the signup button, the application will visit IMDB website and fetch all
   the award winning movies from 1995 to the latest date. The user name and the scraped data will be
   stored in the session variable to improve the performance
4. User will enter the movie title from the form
5. After that, all the details of that user-selected movie will be displayed in the UI
6. When user will click on the logout button, the session variable will get destroyed and 
   the user will be redirected to the signup page again.
